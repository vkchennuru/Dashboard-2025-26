## 🎯 About This Project

This dashboard was developed as part of the **Annual Academic Audit (AAA) 2025-26** initiative to provide a comprehensive, visually engaging, and data-rich overview of all institutional activities conducted during the academic year. It serves as a digital register for:

- 📋 **IQAC Activities** — Internal Quality Assurance Cell programmes
- 🏫 **College Activities** — Department-wise academic and co-curricular events
- 🌍 **Extension Activities** — Community outreach, NSS, SASA, health camps

## 🚀 Getting Started

### Option 1 — Direct Browser Open
```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/skr-activity-dashboard.git

# Open in your browser
open index\(1\).html
```

### Option 2 — GitHub Pages (Recommended)
1. Go to **Settings → Pages**
2. Set source branch to `main`, folder to `/ (root)`
3. Your dashboard will be live at `https://YOUR-USERNAME.github.io/skr-activity-dashboard/`

### Option 3 — Local Server
```bash
# Python (any OS)
python -m http.server 8080

# Then visit
http://localhost:8080/index(1).html
```

## 🛠️ Technologies Used

<div align="center">

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Font Awesome](https://img.shields.io/badge/Font%20Awesome-528DD7?style=for-the-badge&logo=fontawesome&logoColor=white)
![Canvas API](https://img.shields.io/badge/Canvas%20API-Charts-00ACC1?style=for-the-badge)
![CSS Grid](https://img.shields.io/badge/CSS%20Grid-Flexbox-D81B60?style=for-the-badge)

</div>

- **HTML5** — Semantic structure, accessibility-ready
- **CSS3** — Custom properties (variables), Grid, Flexbox, animations, media queries
- **Vanilla JavaScript** — Tab switching, animated counters, live canvas charts, search & filter
- **Canvas API** — Custom bar, donut, and stacked bar charts (no Chart.js dependency)
- **Font Awesome 6.5** — Icon set (loaded via CDN)
- **Google-compatible fonts** — Segoe UI with system fallbacks
